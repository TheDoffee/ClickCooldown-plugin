ClickCooldown - plugin for setting cooldowns on activators

How to use:
1. Aim at an activator (button, pressure plate, lever)
2. Enter the command /clickcooldown <seconds>
3. Done! When the activator is used, it disappears for the specified time

Permissions:
- clickcooldown.use - permission to set cooldowns

Minecraft Version: 1.12.2
